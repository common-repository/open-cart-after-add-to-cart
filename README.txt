This light plugin allows you automatically open the mini-cart when a product added to the cart.

== Description ==

Open Cart After Add to Cart plugin was created for increasing UX on to e-commerce projects based on WooCommerce. After activating the plugin, when the user clicks on the add to cart button, the mini cart opens automatically. Thus, notifying the user that the product was added successfully and can he either make a payment or continue shopping.

== Installation ==

1. Unzip the downloaded zip file and upload the plugin folder into the `wp-content/plugins/` directory. Alternatively, upload from Plugins >> Add New >> Upload Plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress
